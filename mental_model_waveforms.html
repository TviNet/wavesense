Summary of Mental Model

Key Insights from Waveform Analysis:
  • Mode 0: Both MatchEq and Match perform equality detection
  • Mode 8: Match inverts to inequality while MatchEq stays equality
  • Mode 1: NAPOT masking effectively ignores low bits based on tdata2 pattern
  • Modes 2&3: Implement proper unsigned relational comparisons
  • Modes 4&5: Enable selective bit masking for half-word comparisons
